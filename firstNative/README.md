<h3>启动 先配置完成后在clone，启动本项目</h3> <br/>


```
yarn || npm install 

react-native run-ios||react-native run-android

yarn start || npm start 

```

**启动完成 ios command+d 打开调试redux， 安卓可以 command+m 打开调试 redux，这样就可以愉快的开发了**
